
# Toolkit AI Guru Inovatif

Aplikasi web ini adalah platform cerdas untuk membantu guru membuat materi ajar dan penilaian adaptif yang selaras dengan Kurikulum Merdeka di Indonesia.

## Fitur Utama

- **Generator Administrasi:** Membuat ATP, Prota, Promes, Modul Ajar, KKTP, & Jurnal Harian.
- **Generator Bank Soal:** Membuat bank soal, kisi-kisi, & rubrik penilaian.
- **Generator Terpusat:** Membuat administrasi dan bank soal sekaligus dari teks buku ajar.
- **Pencarian Cerdas:** Mendapatkan jawaban akurat dari web & peta terkini.
- **Studio AI:** Alat kreatif untuk membuat gambar, video, dan audio.

## Cara Deploy ke GitHub Pages (Gratis)

Anda dapat dengan mudah men-hosting aplikasi ini secara online dan gratis menggunakan GitHub Pages. Ikuti langkah-langkah berikut:

### Langkah 1: Buat Repository di GitHub

1.  Buka [GitHub](https://github.com/) dan login.
2.  Klik tombol **+** di pojok kanan atas, lalu pilih **New repository**.
3.  Beri nama repository Anda (contoh: `toolkit-ai-guru`).
4.  Pastikan repository diatur ke **Public**.
5.  Klik **Create repository**.

### Langkah 2: Unggah File Aplikasi

1.  Di halaman repository Anda yang baru, klik tombol **Add file**, lalu pilih **Upload files**.
2.  Seret dan lepas (drag and drop) **semua file** dari proyek ini (termasuk `index.html`, `README.md`, dan semua file `.js` di dalam folder `components` dan `services`) ke area unggah.
3.  Setelah semua file selesai diunggah, klik **Commit changes**.

### Langkah 3: Aktifkan GitHub Pages

1.  Di halaman repository Anda, klik tab **Settings**.
2.  Di menu sebelah kiri, klik **Pages**.
3.  Di bawah bagian "Build and deployment", pada "Source", pilih **Deploy from a branch**.
4.  Pada menu dropdown "Branch", pilih **main** dan biarkan folder diatur ke **/(root)**.
5.  Klik **Save**.

Tunggu beberapa menit. GitHub akan membangun situs Anda. Setelah selesai, Anda akan melihat tautan hijau di bagian atas halaman Pages dengan tulisan "Your site is live at...".

**Selesai! Aplikasi Anda sekarang sudah online.**
